# 🎓 College Alumni Network

A full-stack web application designed to connect students and alumni, enabling networking, mentorship, and collaboration.


## 🚀 Features

* 👤 User Authentication (Login/Register)
* 🎓 Alumni & Student Profiles
* 💬 Communication & Networking
* 📢 Announcements / Updates
* 🔍 Search & Connect with Alumni


## 🛠️ Tech Stack

### Frontend

* React.js
* HTML, CSS, JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/gayathriyatheesh/College-Alumni-Network.git
cd College-Alumni-Network
```

---

### 2. Setup Backend

```
cd alumni-backend
npm install
npm run dev
```

---

### 3. Setup Frontend

```
cd client
npm install
npm run dev
```
